    ## JavaScript Quick Start

    Build a search UI with vanilla JavaScript.

    <Steps>
      <Step title="Import the required packages">
        Create your JavaScript file and import InstantSearch:

        ```javascript
        import { liteClient as algoliasearch } from 'algoliasearch/lite';
        import instantsearch from 'instantsearch.js';
        import { searchBox, hits, refinementList, pagination, configure } from 'instantsearch.js/es/widgets';
        import 'instantsearch.css/themes/satellite.css';
        ```

        <Tip>
          We're using `algoliasearch/lite` which is a lighter version of the client optimized for search-only operations.
        </Tip>
      </Step>

      <Step title="Initialize the search client">
        Create an Algolia search client with your credentials:

        ```javascript
        const searchClient = algoliasearch(
          'YourApplicationID',
          'YourSearchOnlyAPIKey'
        );
        ```

        <Warning>
          Replace `YourApplicationID` and `YourSearchOnlyAPIKey` with your actual Algolia credentials. Never use your Admin API Key in client-side code!
        </Warning>
      </Step>

      <Step title="Create the InstantSearch instance">
        Initialize InstantSearch with your index name:

        ```javascript
        const search = instantsearch({
          indexName: 'your_index_name',
          searchClient,
          insights: true,
        });
        ```
      </Step>

      <Step title="Add widgets to your search">
        Add UI widgets to display the search interface:

        ```javascript
        search.addWidgets([
          // Search box
          searchBox({
            container: '#searchbox',
            placeholder: 'Search for products...',
          }),

          // Search results
          hits({
            container: '#hits',
            templates: {
              item: (hit, { html, components }) => html`
                <article>
                  <h2>${components.Highlight({ hit, attribute: 'name' })}</h2>
                  <p>${components.Highlight({ hit, attribute: 'description' })}</p>
                </article>
              `,
            },
          }),

          // Configure search parameters
          configure({
            hitsPerPage: 12,
          }),

          // Category filter
          refinementList({
            container: '#categories',
            attribute: 'categories',
          }),

          // Pagination
          pagination({
            container: '#pagination',
          }),
        ]);
        ```
      </Step>

      <Step title="Start the search">
        Call `start()` to render the search UI:

        ```javascript
        search.start();
        ```
      </Step>

      <Step title="Add HTML containers">
        Create the HTML structure with container elements:

        ```html
        <!DOCTYPE html>
        <html lang="en">
        <head>
          <meta charset="UTF-8">
          <meta name="viewport" content="width=device-width, initial-scale=1.0">
          <title>InstantSearch.js App</title>
        </head>
        <body>
          <header>
            <h1>Search Products</h1>
            <div id="searchbox"></div>
          </header>

          <main>
            <aside>
              <h3>Categories</h3>
              <div id="categories"></div>
            </aside>

            <div>
              <div id="hits"></div>
              <div id="pagination"></div>
            </div>
          </main>

          <script type="module" src="/src/app.js"></script>
        </body>
        </html>
        ```
      </Step>
    </Steps>

    ### Complete Example

    Here's the complete JavaScript code:

    ```javascript
    import { liteClient as algoliasearch } from 'algoliasearch/lite';
    import instantsearch from 'instantsearch.js';
    import { 
      searchBox, 
      hits, 
      refinementList, 
      pagination, 
      configure 
    } from 'instantsearch.js/es/widgets';
    import 'instantsearch.css/themes/satellite.css';

    const searchClient = algoliasearch(
      'YourApplicationID',
      'YourSearchOnlyAPIKey'
    );

    const search = instantsearch({
      indexName: 'your_index_name',
      searchClient,
      insights: true,
    });

    search.addWidgets([
      searchBox({
        container: '#searchbox',
        placeholder: 'Search for products...',
      }),

      hits({
        container: '#hits',
        templates: {
          item: (hit, { html, components }) => html`
            <article>
              <h2>${components.Highlight({ hit, attribute: 'name' })}</h2>
              <p>${components.Highlight({ hit, attribute: 'description' })}</p>
            </article>
          `,
        },
      }),

      configure({
        hitsPerPage: 12,
      }),

      refinementList({
        container: '#categories',
        attribute: 'categories',
      }),

      pagination({
        container: '#pagination',
      }),
    ]);

    search.start();
    ```

    ## React Quick Start

    Build a search UI with React components.

    <Steps>
      <Step title="Import the required packages">
        Create your React component and import InstantSearch:

        ```tsx
        import { liteClient as algoliasearch } from 'algoliasearch/lite';
        import React from 'react';
        import {
          InstantSearch,
          SearchBox,
          Hits,
          RefinementList,
          Pagination,
          Highlight,
          Configure,
        } from 'react-instantsearch';
        import 'instantsearch.css/themes/satellite.css';
        ```
      </Step>

      <Step title="Initialize the search client">
        Create an Algolia search client with your credentials:

        ```tsx
        const searchClient = algoliasearch(
          'YourApplicationID',
          'YourSearchOnlyAPIKey'
        );
        ```

        <Warning>
          Replace `YourApplicationID` and `YourSearchOnlyAPIKey` with your actual Algolia credentials. Never use your Admin API Key in client-side code!
        </Warning>
      </Step>

      <Step title="Create the App component">
        Build your search interface using React components:

        ```tsx
        export function App() {
          return (
            <InstantSearch
              searchClient={searchClient}
              indexName="your_index_name"
              insights={true}
            >
              <Configure hitsPerPage={12} />
              
              <header>
                <h1>Search Products</h1>
                <SearchBox placeholder="Search for products..." />
              </header>

              <main>
                <aside>
                  <h3>Categories</h3>
                  <RefinementList attribute="categories" />
                </aside>

                <div>
                  <Hits hitComponent={Hit} />
                  <Pagination />
                </div>
              </main>
            </InstantSearch>
          );
        }
        ```
      </Step>

      <Step title="Create the Hit component">
        Define how individual search results should be displayed:

        ```tsx
        function Hit({ hit }) {
          return (
            <article>
              <h2>
                <Highlight attribute="name" hit={hit} />
              </h2>
              <p>
                <Highlight attribute="description" hit={hit} />
              </p>
            </article>
          );
        }
        ```
      </Step>

      <Step title="Render your app">
        Mount your app to the DOM:

        ```tsx
        import { createRoot } from 'react-dom/client';

        const root = createRoot(document.getElementById('root'));
        root.render(<App />);
        ```
      </Step>
    </Steps>

    ### Complete Example

    Here's the complete React component:

    ```tsx
    import { liteClient as algoliasearch } from 'algoliasearch/lite';
    import React from 'react';
    import {
      InstantSearch,
      SearchBox,
      Hits,
      RefinementList,
      Pagination,
      Highlight,
      Configure,
    } from 'react-instantsearch';
    import { Hit } from 'instantsearch.js';
    import 'instantsearch.css/themes/satellite.css';

    const searchClient = algoliasearch(
      'YourApplicationID',
      'YourSearchOnlyAPIKey'
    );

    export function App() {
      return (
        <InstantSearch
          searchClient={searchClient}
          indexName="your_index_name"
          insights={true}
        >
          <Configure hitsPerPage={12} />
          
          <header>
            <h1>Search Products</h1>
            <SearchBox placeholder="Search for products..." />
          </header>

          <main>
            <aside>
              <h3>Categories</h3>
              <RefinementList attribute="categories" />
            </aside>

            <div>
              <Hits hitComponent={HitComponent} />
              <Pagination />
            </div>
          </main>
        </InstantSearch>
      );
    }

    type HitType = Hit<{
      name: string;
      description: string;
    }>;

    function HitComponent({ hit }: { hit: HitType }) {
      return (
        <article>
          <h2>
            <Highlight attribute="name" hit={hit} />
          </h2>
          <p>
            <Highlight attribute="description" hit={hit} />
          </p>
        </article>
      );
    }
    ```

    ### TypeScript Support

    React InstantSearch has full TypeScript support. Type your hit data for better autocomplete:

    ```tsx
    import { Hit } from 'instantsearch.js';

    type ProductHit = Hit<{
      name: string;
      description: string;
      price: number;
      image: string;
      categories: string[];
    }>;

    function HitComponent({ hit }: { hit: ProductHit }) {
      // TypeScript knows about all your hit properties
      return (
        <article>
          <img src={hit.image} alt={hit.name} />
          <h2><Highlight attribute="name" hit={hit} /></h2>
          <p>{hit.price}</p>
        </article>
      );
    }
    ```

    ## Vue Quick Start

    Build a search UI with Vue components.

    <Steps>
      <Step title="Register the InstantSearch plugin">
        In your main.js, register Vue InstantSearch:

        ```javascript
        import Vue from 'vue';
        import InstantSearch from 'vue-instantsearch';
        import App from './App.vue';

        Vue.config.productionTip = false;
        Vue.use(InstantSearch);

        new Vue({
          render: (h) => h(App),
        }).$mount('#app');
        ```
      </Step>

      <Step title="Create the search client">
        In your App.vue, import and create the search client:

        ```vue
        <script>
        import { liteClient as algoliasearch } from 'algoliasearch/lite';

        export default {
          data() {
            return {
              searchClient: algoliasearch(
                'YourApplicationID',
                'YourSearchOnlyAPIKey'
              ),
            };
          },
        };
        </script>
        ```

        <Warning>
          Replace `YourApplicationID` and `YourSearchOnlyAPIKey` with your actual Algolia credentials. Never use your Admin API Key in client-side code!
        </Warning>
      </Step>

      <Step title="Build the template">
        Use InstantSearch Vue components in your template:

        ```vue
        <template>
          <div>
            <ais-instant-search
              :search-client="searchClient"
              index-name="your_index_name"
              :insights="true"
            >
              <ais-configure :hits-per-page.camel="12" />
              
              <header>
                <h1>Search Products</h1>
                <ais-search-box placeholder="Search for products..." />
              </header>

              <main>
                <aside>
                  <h3>Categories</h3>
                  <ais-refinement-list attribute="categories" />
                </aside>

                <div>
                  <ais-hits>
                    <template slot="item" slot-scope="{ item }">
                      <article>
                        <h2>
                          <ais-highlight :hit="item" attribute="name" />
                        </h2>
                        <p>
                          <ais-highlight :hit="item" attribute="description" />
                        </p>
                      </article>
                    </template>
                  </ais-hits>

                  <ais-pagination />
                </div>
              </main>
            </ais-instant-search>
          </div>
        </template>
        ```
      </Step>

      <Step title="Add styles">
        Import the InstantSearch CSS theme:

        ```vue
        <style>
        @import 'instantsearch.css/themes/satellite.css';

        /* Your custom styles */
        main {
          display: flex;
          gap: 2rem;
        }

        aside {
          flex: 1;
        }

        aside + div {
          flex: 3;
        }
        </style>
        ```
      </Step>
    </Steps>

## What's Happening?

Let's break down the key parts:

1. **Search Client** - Connects to your Algolia application using the lite client (optimized for search)
2. **InstantSearch Instance** - The main container that manages the search state
3. **Widgets/Components** - UI elements that render and interact with the search:
   - `SearchBox` - Input field for search queries
   - `Hits` - Displays search results
   - `RefinementList` - Faceted filter for attributes
   - `Pagination` - Navigate through pages
   - `Configure` - Set search parameters
4. **Highlight** - Highlights matching terms in results

## Troubleshooting

### Search Results Not Appearing

- Verify your Application ID and API Key are correct
- Check that your index name matches exactly
- Ensure your index has data
- Check the browser console for errors

### Widgets Not Rendering

- Verify HTML containers exist (JavaScript version)
- Check that you called `search.start()` (JavaScript version)
- Ensure the InstantSearch component wraps all widgets (React/Vue)

### TypeScript Errors

- Install the correct type packages for your Algolia client version
- Import the `Hit` type from `instantsearch.js`
- Type your hit data properly
