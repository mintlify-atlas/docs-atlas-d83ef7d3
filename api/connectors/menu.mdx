---
title: "connectMenu"
description: "Build a custom widget for single-select faceted filtering"
---

The **Menu** connector provides the logic to build a custom widget that lets users choose a single value for a specific facet. This is typically used for navigation in categories.

<Note>
  The attribute passed as `attribute` must be present in **attributesForFaceting** on the Algolia dashboard or configured via the API.
</Note>

## Usage

```javascript
import { connectMenu } from 'instantsearch.js/es/connectors';

const customMenu = connectMenu(
  (renderOptions, isFirstRender) => {
    const { items, refine, widgetParams } = renderOptions;
    const { container } = widgetParams;
    
    container.innerHTML = `
      <ul>
        ${items.map(item => `
          <li>
            <a 
              href="#" 
              data-value="${item.value}"
              style="${item.isRefined ? 'font-weight: bold' : ''}"
            >
              ${item.label} (${item.count})
            </a>
          </li>
        `).join('')}
      </ul>
    `;
    
    container.querySelectorAll('a').forEach(link => {
      link.addEventListener('click', (e) => {
        e.preventDefault();
        refine(e.target.dataset.value);
      });
    });
  }
);

search.addWidgets([
  customMenu({
    container: document.querySelector('#menu'),
    attribute: 'category',
  }),
]);
```

## Connector Options

<ParamField path="attribute" type="string" required>
  Name of the attribute for faceting (e.g., "category").
</ParamField>

<ParamField path="limit" type="number" default="10">
  How many facet values to retrieve.
</ParamField>

<ParamField path="showMore" type="boolean" default="false">
  Whether to display a button that expands the number of items.
</ParamField>

<ParamField path="showMoreLimit" type="number" default="20">
  How many facet values to retrieve when `toggleShowMore` is called.
</ParamField>

<ParamField path="sortBy" type="string[] | function" default="['isRefined', 'name:asc']">
  How to sort refinements. Possible values: `count`, `isRefined`, `name:asc`, `name:desc`.
  
  You can also use a sort function that behaves like the standard JavaScript [compareFunction](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Array/sort#Syntax).
</ParamField>

<ParamField path="transformItems" type="(items: object[]) => object[]">
  Function to transform the items passed to the templates.
</ParamField>

## Render Options

<ParamField path="items" type="Array<MenuItem>">
  The elements that can be refined for the current search results.
  
  Each item has:
  - `value`: The value of the menu item
  - `label`: Human-readable value of the item
  - `count`: Number of results matched after refinement
  - `isRefined`: Indicates if the refinement is applied
</ParamField>

<ParamField path="refine" type="(value: string) => void">
  Filters the search to the item value. Selecting a new item will unselect the current one.
</ParamField>

<ParamField path="createURL" type="(value: string) => string">
  Creates the URL for a single item name in the list.
</ParamField>

<ParamField path="canRefine" type="boolean">
  `true` if refinement can be applied.
</ParamField>

<ParamField path="isShowingMore" type="boolean">
  `true` if the menu is displaying all items.
</ParamField>

<ParamField path="toggleShowMore" type="() => void">
  Toggles the number of values displayed between `limit` and `showMoreLimit`.
</ParamField>

<ParamField path="canToggleShowMore" type="boolean">
  `true` if the toggleShowMore button can be activated.
</ParamField>

<ParamField path="sendEvent" type="function">
  Sends an event to the Insights middleware.
</ParamField>

<ParamField path="widgetParams" type="object">
  The options passed to the connector.
</ParamField>
