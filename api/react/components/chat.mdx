---
title: '<Chat>'
description: 'AI-powered conversational search interface'
---

The `<Chat>` component provides an AI-powered conversational search interface for interacting with your search results using natural language.

## Import

```jsx
import { Chat } from 'react-instantsearch';
```

## Props

<ParamField path="title" type="string">
  The title displayed in the chat header.

  ```jsx
  <Chat title="Ask about our products" />
  ```
</ParamField>

<ParamField path="tools" type="UserClientSideTools">
  Custom tools for the chat agent to use.

  ```jsx
  import { SearchIndexToolType } from 'react-instantsearch';

  <Chat
    tools={{
      [SearchIndexToolType]: {
        // custom tool configuration
      },
    }}
  />
  ```
</ParamField>

<ParamField path="itemComponent" type="React.Component">
  Component to render individual items in search results.

  ```jsx
  <Chat
    itemComponent={({ item }) => (
      <div>
        <h4>{item.name}</h4>
        <p>{item.description}</p>
      </div>
    )}
  />
  ```
</ParamField>

<ParamField path="getSearchPageURL" type="function">
  Function to generate URLs for search result pages.

  ```jsx
  <Chat
    getSearchPageURL={(uiState) => `/search?q=${uiState.query}`}
  />
  ```
</ParamField>

<ParamField path="toggleButtonProps" type="object">
  Props to customize the toggle button.
</ParamField>

<ParamField path="headerProps" type="object">
  Props to customize the chat header.
</ParamField>

<ParamField path="messagesProps" type="object">
  Props to customize the messages area.
</ParamField>

<ParamField path="promptProps" type="object">
  Props to customize the prompt input area.
</ParamField>

<ParamField path="toggleButtonComponent" type="React.Component">
  Custom component for the toggle button.
</ParamField>

<ParamField path="headerComponent" type="React.Component">
  Custom component for the chat header.
</ParamField>

<ParamField path="promptComponent" type="React.Component">
  Custom component for the prompt input.
</ParamField>

<ParamField path="suggestionsComponent" type="React.Component">
  Custom component for displaying suggestions.
</ParamField>

<ParamField path="translations" type="object">
  Customize the text strings.

  ```jsx
  <Chat
    translations={{
      prompt: { placeholder: 'Ask me anything...' },
      header: { title: 'Search Assistant' },
    }}
  />
  ```
</ParamField>

<ParamField path="classNames" type="object">
  CSS classes to customize the component styling.
</ParamField>

## Example

```jsx
import { InstantSearch, Chat } from 'react-instantsearch';
import { liteClient as algoliasearch } from 'algoliasearch/lite';

const searchClient = algoliasearch(
  'YourApplicationID',
  'YourSearchOnlyAPIKey'
);

function App() {
  return (
    <InstantSearch searchClient={searchClient} indexName="products">
      <Chat
        title="Product Assistant"
        itemComponent={({ item }) => (
          <div>
            <img src={item.image} alt={item.name} />
            <h4>{item.name}</h4>
            <p>${item.price}</p>
          </div>
        )}
      />
    </InstantSearch>
  );
}
```
